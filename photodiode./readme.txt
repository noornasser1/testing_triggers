//Photodiode
1.Connect the photodiode to the EEG auxillary box (see the pins and be gentle)
2.Paste the photodiode to the PC screen
3.Change the PC screen input to the stimulus PC
4.Enter the EEGO software so that you'll see the photodiode registering the signal from the diode
5.run diode.py which can be found in the projects folder under "Testing latency with photodiode"
6.Run the script in which the screen window will change from black to white
7.Run photodiode_lsl_triggers to see the data of the photodiode and the lsl trigger combined
